## 🎯 Common Tasks

### Task: "I want to see the new design"
1. Open `modern-style-preview.html` in browser
2. Browse all components
3. Compare with current site

### Task: "I need to update a component"
1. Check `QUICK_REFERENCE.md` for the component
2. Copy the example code
3. Reference `MODERN_STYLE_GUIDE.md` for details

### Task: "I need to test the migration"

3. Document and report
