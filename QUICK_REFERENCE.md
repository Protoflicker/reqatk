# 🚀 Quick Reference - Modern Style

## Buttons
```html
<button class="neu-btn">Default</button>
<button class="neu-btn-primary">Primary</button>
<button class="neu-btn-danger">Delete</button>
<button class="neu-btn"><span class="icon icon-sm">➕</span> Add</button>
```

## Inputs
```html
<label class="label">Field Name</label>
<input type="text" class="neu-input" placeholder="Enter...">
```

## Cards
```html
<div class="card">Content</div>
<div class="card card-compact">Compact</div>
<div class="card-primary">Highlighted</div>
```

## Badges
```html
<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-danger">Danger</span>
```

## Grids
```html
<div class="grid-responsive">...</div>
<div class="grid-2">...</div>
<div class="grid-3">...</div>
```

## Status
```html
<span class="status status-success">Active</span>
<span class="status status-danger">Inactive</span>
```

## Avatars
```html
<div class="avatar avatar-md">JD</div>
```

## Icons
```html
<span class="icon icon-sm">🔍</span>
<span class="icon icon-lg">⚙️</span>
```

## Animations
```html
<div class="animate-fade-up">...</div>
<div class="animate-fade-in">...</div>
```

## Loading
```html
<div class="skeleton" style="height: 100px;"></div>
```

## Divider
```html
<div class="divider"></div>
```

## Table
```html
<table class="tbl">
  <thead><tr><th>Header</th></tr></thead>
  <tbody><tr><td>Cell</td></tr></tbody>
</table>
```

---

## CSS Variables

```css
/* Radius */
--radius-input: 6px
--radius: 8px
--radius-lg: 12px

/* Colors */
--color-primary: #0075de
--color-success: #1aae39
--color-danger: #e03e3e

/* Shadows */
--shadow-card
--shadow-hover
--shadow-lg
```

---

## Responsive

```css
@media (max-width: 768px) {
  /* Mobile styles */
}
```

Grids automatically collapse to 1 column on mobile.
